## Game Show Theme Ideas

### Theme Concept
The game is a deadly game show where contestants fight through studio floors. As they progress, they encounter increasingly important show staff and executives.

### Naming Progression Pattern
- **Floor 1**: Entry-level staff (Security, Fan, Extra, etc.)
- **Floor 2**: Mid-level staff (Head of Security, Groupie, Actor/Actress, etc.)
- **Floor 3**: Senior staff (Director, Producer, etc.)
- **Floor 4**: Executive level (Showrunner, Executive Producer, etc.)

### Suggested Enemy Re-theming

#### Floor 1 - Entry Level
- **rusher** → **Security Guard** (runs at contestant)
- **shooter** → **Camera Operator** (ranged attacks)
- **zombie** → **Audience Member** (slow, persistent)
- **slime** → **Fan** (weak, splits into more fans)
- **bat** → **Stagehand** (fast, erratic movement)

#### Floor 2 - Mid Level
- **charger** → **Head of Security** (charges at contestant)
- **turret** → **Camera Director** (stationary, shoots)
- **heavyTank** → **Bouncer** (tanky, slow)
- **zippy** → **Runner** (extremely fast)
- **exploder** → **Pyrotechnics Tech** (explodes)
- **orbiter** → **Floor Manager** (patrols perimeter, dangerous)

#### Floor 3 - Senior Staff
- **mage** → **Lighting Director** (powerful ranged)
- **shieldBearer** → **Stage Manager** (protects others)
- **golem** → **Set Designer** (tanky, builds obstacles)
- **wraith** → **Editor** (teleports, cuts between shots)
- **spawner** → **Casting Director** (spawns more enemies)
- **buffer** → **Assistant Director** (buffs nearby enemies)

#### Floor 4 - Executive Level
- **healer** → **Producer** (supports team)
- **teleporter** → **Executive Producer** (teleports player)
- **freezer** → **Network Executive** (slows player)
- **leech** → **Talent Agent** (lifesteal, takes from player)

### Suggested Mini-Boss Re-theming

1. **brute** → **Security Chief** (charges, heavy armor)
2. **sentinel** → **Technical Director** (ranged, shields)
3. **berserker** → **Stunt Coordinator** (fast, aggressive)
4. **guardian** → **Studio Manager** (tanky, protective)
5. **warlock** → **Creative Director** (ranged, shields, regen)

### Suggested Boss Re-theming

1. **gatekeeper** → **Studio Head** (first boss, balanced)
2. **swarmQueen** → **Showrunner** (spawns enemies, tanky)
3. **twinGuardianMelee** → **Co-Host (Melee)** (fast melee twin)
4. **twinGuardianRanged** → **Co-Host (Ranged)** (ranged twin)
